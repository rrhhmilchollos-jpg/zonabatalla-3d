# ZonaBatalla 3D

Juego web 3D de batalla campal inspirado en el género battle royale: explora un mapa, elimina rivales IA, recoge recursos y sobrevive en la zona segura. Diseñado para jugadores casuales en navegador móvil y escritorio.

Generado y mantenido automáticamente por **Maris AI**. Cada vez que actualizas la app desde Maris AI, este repo recibe un commit nuevo en `main` con la versión actual del código.

## Estructura

- `frontend/` — React + Vite + Tailwind

## Cómo correrlo en local

```bash
cd frontend
npm install
npm run dev
```
